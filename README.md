Aplicación Android desarrollada con Kotlin, Jetpack Compose, siguiendo principios de Clean Architecture y el patrón Model–View–ViewModel (MVVM).

Principios aplicados:

Separación de responsabilidades
Inversión de dependencias
Estado reactivo con ViewModel
UI declarativa con Compose
Manejo de estado con StateFlow / Flow

Requisitos
Antes de compilar asegúrate de tener:

Android Studio Hedgehog o superior
JDK 17
Gradle 8+
SDK mínimo definido en el proyecto (ej: minSdk 24)
Dispositivo físico o emulador configurado

Cómo compilar el proyecto?

Clona el repositorio
Abre el proyecto en Android Studio.
Espera a que Gradle sincronice dependencias.
Si es necesario:
File → Sync Project with Gradle Files

Cómo ejecutar la aplicación?

Inicia un emulador o conecta un dispositivo físico.
Selecciona el módulo app.
Presiona Run (▶)
